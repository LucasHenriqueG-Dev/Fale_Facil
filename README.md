# Fale_Facil
O sistema Fale Fácil é uma solução digital desenvolvida para otimizar a gestão de demandas internas na empresa PAREX Indústria Mecânica, Caldeiraria e Estruturas Metálicas S.A. Atualmente, o processo manual utilizado gera problemas como retrabalho, perda de informações e baixa eficiência no acompanhamento das solicitações.
